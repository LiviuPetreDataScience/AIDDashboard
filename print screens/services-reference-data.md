## Services table columns (header)
1. Path (Service Category — hierarchy, shown bundled on input)
2. Services (the service name)
3. Contract end date (SOW) — date
4. Does this service exist in the company? — Yes/No
5. Is it provided by Stefanini Group? — Yes/No
6. If not provided by SG then by who (by client or by a competitor provider)? — Competitor dropdown
7. If service is not existent, is there an opportunity to provide it? — Yes/No
8. Details — text
